Git Lesson Notes
git status: checks changes, new file and folder creations.
git add [filename]: add files to staging status
git add .: add all files on root directory
git commit -m [Commit Message] [filename]: commits files
git rm --cached [filename]: remove file from staging
git diff [filename]: shows difference in files versions
git log: lists the commitments that have been made
git reset --hard [xxxxxx]: reset to previous commit by using first six digits of commit Key
